This is a simple quest helper tool for Escape From Tarkov that allows for dynamic showing of quest locations and objectives on a per map basis

### Massive thanks to the Tarkov wiki maintainers and map creators as this would not have been possible without them.

## USE

Quests can be moved between 'inactive', 'active', and 'completed' states by selecting the quest manager, then right clicking the quest and selecting an option.

Quests can be toggled on and off on a map by checking/dechecking the box next to the main quest name on the upper right panel.

Quest objectives can be viewed by expanding the quest submenu in the lower right panel

## How 2 run

- Have python installed

- pip install dependencies (pillow)

- either run using `python main.py` or by double clicking the `main.py` file in your file browser

## Requirements

- Python (coded in 3.12.3, might work on others idk)

- pillow (11.3 used, others might work again idk)
